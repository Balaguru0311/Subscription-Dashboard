# Subscription-Dashboard


Tech Stack
- Frontend: React + Vite + Tailwind + Redux Toolkit
- Backend: Node.js + Express + MongoDB (Mongoose)
- Auth: JWT + Refresh Tokens

Setup
Backend
```bash
cd server
npm install
cp .env.example .env

npm run dev

Frontend

npm run dev
